# 🚀 快速入门指南

## 立即开始使用

### 1. 安装依赖 (约1分钟)
```bash
pip install -r requirements.txt
```

### 2. 运行购物助手
```bash
python main.py
```

### 3. 选择AI服务
- 直接按回车使用默认Qwen服务 ✅ 
- 或选择OpenAI服务
- API密钥已内置，无需配置 🎉

### 4. 开始购物咨询
```
您: 我想在北京买一部手机，预算5000元
🤖: 正在为您搜索...

您: 帮我找找上海的笔记本电脑
🤖: 为您推荐以下商品...
```
